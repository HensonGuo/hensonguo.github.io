双星号指针

2022-10-27 1 min read # C++
  1. 指针本身就是一个对象，对象存储的是另一个变量的地址
  2. &ptr，获取的是指针对象的地址，而非存储变量的地址
int a = 1;
int* b;
b = &a;
int c = *b;
int d = **(&b);

以上bcd的值都为1
&b 指针地址；*(&b) 指针；**(&b) 指针变量

指针引用作为函数参数传递

void getData(int**a, int& b)
{
	int x = 1;
	int y = 2;
	int* test = &x;
	*a = test;
	b = y;
}

int main()
{
	int* m = nullptr;
	int n = 0;
	getData(&m, n);
	printf("%d, %d", *m, n);
	return 0;
}

1、指针本身是一个对象，将指针地址传入&m
2、指针的指针加双星号**a
3、更改指针地址则是 *a = ptr

    • 指针引用作为函数参数传递
