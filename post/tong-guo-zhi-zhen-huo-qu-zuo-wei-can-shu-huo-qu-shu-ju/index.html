通过指针获取作为参数获取数据

2022-10-27 1 min read # C++ 
void getData(int**a, int& b)
{
	int x = 1;
	int y = 2;
	int* test = &x;
	*a = test;
	b = y;
}

int main()
{
	int* m = nullptr;
	int n = 0;
	getData(&m, n);
	printf("%d, %d", *m, n);
	return 0;
}

1、指针本身是一个对象，将指针地址传入&m
2、指针的指针加双星号**a
3、更改指针地址则是 *a = ptr
