git子仓库设置与获取

2022-08-22 1 min read # git

  1. 添加子仓库

    git submodule add git@xxx.git

    项目目录下会生成.gitmodules文件，其文件格式为

    [submodule "apps"]
     path = dirpath
     url = ssh://git@xxxx.git
     branch = branchname
     ignore = all

    其中path为项目的子仓库存储目录
    url为对应的远程仓库地址

  2. 删除子仓库

    cd .git/modules 
rm -rf xxxx

vi .git/config

  3. 子仓库获取1
    git clone --recursive ssh://xxx.git
